# To 24 hours

In a file `to24h.q` write a function `to24h`:

	q)s:("12:05.07 AM";"05:30:12 AM";"12:02:23 PM";"02:41:23 PM";"09:51:02 PM";"11:59:37 PM")
	q)s!to24h s
	"12:05.07 AM"| 00:05:07
	"05:30:12 AM"| 05:30:12
	"12:02:23 PM"| 12:02:23
	"02:41:23 PM"| 14:41:23
	"09:51:02 PM"| 21:51:02
	"11:59:37 PM"| 23:59:37
